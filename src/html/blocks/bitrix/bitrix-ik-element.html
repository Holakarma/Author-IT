Все описанные результаты нашей работы позволили ООО «Теплосервис-ТС» двигаться огромными шагами к новым высотам.

Устранить недоработки корпоративного портала Битрикс24. Расширить связь между интернет-магазином и CRM-системой. Внедрить и настроить технологию, позволяющую гибко использовать бонусную программу. Создать удобную среду для HR-ов для найма новых сотрудников.

Внедрена и настроена бонусная программа, которая позволяет клиентам накапливать и тратить бонусы, оплачивая ими товары. Отдел найма больше не перепрыгивает между страничками Авито и hh.ru в поисках новых сотрудников. Теперь вся работа ведётся внутри Битрикс24.
